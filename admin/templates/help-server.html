安裝步驟:
1. 安裝 centos7.5、 mysql、php7、nginx
2. 如果mysql沒有密碼，則可以執行 mysql -uroot < install/mysql.sql 會自動建立一個 ipv6ddns 資料庫
3. 如果mysql有密碼，請執行 mysql -uroot -p < install/mysql.sql 會出現提示訊息，請輸入你的密碼
4. 下載php程式碼之後，要安裝php環境請在程式碼目錄下執行 composer install
5. 裝好環境之後，可以執行 http://localhost/admin/
6. 預設內建一個帳號，帳號為 admin 密碼為 w123456
7. 登入之後，請記得更改密碼
需要啟動排成(cron job)，定期更新Server的IP給Client使用。
執行方式為設定cron job，如果不知道何為cron job，請用Google搜尋 crontab。
0 * * * * /usr/bin/php /var/www/ddnsipv6/cron/update-ip.php 2>&1 > /dev/null
操作步驟：
1. 先新增網域
2. 再新增主機(Hostname)
3. 最後才是新增API Key
